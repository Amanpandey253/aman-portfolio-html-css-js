# Responsive Portfolio Website

A responsive personal portfolio website built using HTML5, CSS3, and JavaScript. The website showcases personal information, skills, projects, and contact details with a modern and user-friendly design.

## Features

* Responsive Design
* Interactive Navigation
* About Section
* Skills Section
* Projects Showcase
* Contact Section
* Hover Effects and Animations

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Flexbox
* CSS Grid
* Media Queries

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/aman-portfolio-html-css-js.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.
